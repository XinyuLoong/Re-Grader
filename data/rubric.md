# Rubric

## Correctness
- Full credit if the palindrome-checking logic correctly determines whether a string reads the same forward and backward.
- Deduct if the method is logically incorrect or incomplete.

## Completeness
- Full credit if the student provides both an implementation idea and a time complexity analysis.
- Deduct if one required component is missing.

## Edge Cases
- Full credit if the student addresses important edge cases such as empty strings, single-character strings, punctuation handling, or case sensitivity when relevant.
- Deduct if no edge-case reasoning is provided.

## Explanation Quality
- Full credit if the explanation is clear and well-structured.
- Deduct if the explanation is vague or hard to follow.
