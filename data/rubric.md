# Rubric (10 points total)

## 1. Functional correctness (4 points)
- 4: Correctly determines whether the string is a palindrome.
- 2-3: Mostly correct but contains minor logical issues.
- 0-1: Incorrect or incomplete logic.

## 2. Handling of case / spaces / punctuation (2 points)
- 2: Correctly ignores case, spaces, and punctuation.
- 1: Handles some but not all normalization requirements.
- 0: Does not address normalization requirements.

## 3. Time complexity explanation (2 points)
- 2: Correct and clearly explains time complexity.
- 1: Mentions complexity but explanation is incomplete or partially incorrect.
- 0: Missing or incorrect complexity discussion.

## 4. Clarity / examples / tests (2 points)
- 2: Clear explanation and includes at least one example or test case.
- 1: Some explanation but limited clarity or no meaningful test/example.
- 0: No explanation or examples/tests.
